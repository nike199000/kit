    palette

    @function palette($name: null, $variant: null) { ... }

    Description

    Generates a color value in HEX or RGBA based on $color-palette map.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $name

    The name of a color

    		Stringnull
    $variant

    The varient of a color

    		Numbernull

    Returns

    Color —

    Value in HEX or RGBA format.

    Example

    body {
  background-color: palette('green');
  color: palette('black', 300);
}

    Throws

    • An error if a color name or varient aren't found.

    Requires

    • [function] get-color
    • [function] get-alpha

    • [mixin] background-color

    Map of varients and their alphas

    @function get-alpha($color, $variant) { ... }

    Description

    Gets the alpha value of a color by it's varient.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $color

    Name of a color to determine which varients map to use.

    		String— none
    $variant

    A number between 100-300 multiples by 100 for colors, and a number between 50-300 multiples by 50 for black and white colors.

    		Number— none

    Returns

    Number —

    A number between 0 to 1.

    Example

    get-alpha('black', 50) // 0.07
